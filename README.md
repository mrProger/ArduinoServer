# ArduinoServer

## Подключение к MEGA2560

### Ethernet Shield
**5V->5V**
**GND->GND**
**MISO->50(SPI MISO Pin)**
**MOSI->51(SPI MOSI Pin)**
**RST->RESET**
**SCS->10(SPI SCS Pin)**
**SCLK->52(SPI SCLK Pin)**

### TFT 1.8 Display
**VCC->5V**
**GND->GND**
**CS->11**
**RESET->9**
**A0->8**
**SDA->38**
**SCK->40**
**LED->3.3V**
